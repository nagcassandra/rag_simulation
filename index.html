RAG Step Journey

  1. Collect Docs - Documents enter the knowledge base.
  2. Chunk Docs - Large text is split into smaller pieces.
  3. Create Embeddings - Each chunk becomes a vector representation.
  4. Vector Search - User prompt is embedded and similar chunks are retrieved.
  5. Generate Answer - LLM uses retrieved chunks as context to answer.
